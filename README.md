## 🛠️ Simulation Core Tasks

### 📂 Task 1: Advanced Data Structures
* **Video Guidance:** Ashley Smith, Tech Campus Recruiter at Walmart

#### Objective
* Solve algorithmic challenges related to Walmart's shipping operations.

#### Implementation
* Developed a custom heap-based data structure (`PowerHeap`) in Java.
* Applied data structure optimization techniques to improve priority-based processing efficiency.

#### Key Concepts
* Heap Data Structure
* Algorithm Design
* Computational Complexity
* Java Programming

---

### 📂 Task 2: Software Architecture
* **Video Guidance:** Brian Eappen, Technical Program Manager at Walmart

#### Objective
* Design a scalable architecture for a production-grade data processor.

#### Implementation
* Created a UML Class Diagram for a Data Processor system.
* Designed abstractions for multiple operating modes and database integrations.
* Modeled interactions with PostgreSQL, Redis, and Elasticsearch.

#### Key Concepts
* UML Modeling
* Object-Oriented Design
* Software Architecture
* System Design

---

### 📂 Task 3: Relational Database Design
* **Video Guidance:** Jonathan, Walmart Student Ambassador

#### Objective
* Design a normalized database schema for Walmart's Pet Department.

#### Implementation
* Created an Entity-Relationship Diagram (ERD).
* Designed entities, relationships, constraints, and database structure based on business requirements.
* Applied database normalization principles.

#### Key Concepts
* ER Modeling
* Relational Database Design
* Database Normalization (3NF)
* Data Modeling

---

### 📂 Task 4: Data Munging & Processing
* **Video Guidance:** Keri Reynolds, Software Engineering Campus Recruiting Lead at Walmart

#### Objective
* Process and transform spreadsheet data for database ingestion.

#### Implementation
* Developed a Python automation script to:
  * Read spreadsheet data
  * Extract relevant information
  * Clean and transform records
  * Insert formatted data into a database

#### Key Concepts
* Python Automation
* Data Processing
* ETL Workflows
* Database Integration
