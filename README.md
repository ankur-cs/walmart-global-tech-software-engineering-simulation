## 📌 Overview
This repository contains the advanced engineering deliverables completed during the **Walmart Global Tech USA Advanced Software Engineering Job Simulation** hosted on Forage. The program involved analyzing, optimizing, and architecting production-grade solutions for a variety of core engineering teams at Walmart. The work spans across high-performance algorithmic data structures, decoupled software architecture design, and comprehensive relational database modeling.

---

## 🛠️ Simulation Core Tasks

### 📂 Task 1: Advanced Data Structures (Algorithmic Optimization)
* **Objective:** Solve complex, high-throughput logistical challenges within Walmart’s distribution pipelines.
* **Implementation:** Developed a novel, optimized version of a custom heap data structure in Java engineered specifically for **Walmart’s Shipping Department**. 
* **Key Focus:** Showcased robust low-level problem-solving and algorithmic mechanics to optimize priority processing and maintain efficient computational complexity bounds.

### 📂 Task 2: Software Architecture (System Design)
* **Objective:** Coordinate a clear, extensible structural blueprint for core backend data utilities.
* **Implementation:** Architected a comprehensive **UML Class Diagram** for a production-grade **Data Processor**.
* **Key Focus:** Designed the system to gracefully handle multiple distinct operating modes while structurally accounting for dynamic downstream database connections.

### 📂 Task 3: Relational Database Design (Data Modeling)
* **Objective:** Design an agile, normalized relational schema tailored to support dedicated, multi-departmental inventory and tracking demands.
* **Implementation:** Engineered a robust **Entity-Relationship Diagram (ERD)** built completely from scratch.
* **Key Focus:** Fully mapped and accounted for all complex architectural data requirements provided by **Walmart’s Pet Department** to enforce absolute transactional integrity.

### 📂 Task 4: Data Munging (Pipeline Processing)
* **Objective:** Cleanse, align, and manipulate raw logistical datasets to prepare disparate files for analytics ingestion pipelines.
* **Implementation:** Built targeted transformations to parse structural formats, manage inconsistencies, and transform input fields into schema-compliant, uniform data formats.

## 🚀 Technical Framework & Core Concepts
* **Languages & Scripting:** Java (JDK 17+)
* **System Design & Modeling:** Unified Modeling Language (UML), Entity-Relationship Diagram (ERD)
* **Computer Science Core:** Custom Data Structures, Algorithmic Optimization, Database Normalization, Software Design Patterns
